13 Claude Skills para Nuvemshop

Cada skill conecta diretamente na API da sua loja e entrega análises em 2 minutos — sem planilha, sem exportação manual.

Claude Code API Nuvemshop Python · curl
Projeto pessoal, não oficial. Estas skills foram criadas por Thiago Rocha e não têm relação com a Nuvemshop — a empresa não oferece suporte a este projeto. Use por sua conta e risco e sempre revise os dados antes de aplicar qualquer alteração na sua loja.
Outputs reais. Todos os resultados abaixo foram gerados via API da Nuvemshop durante os testes de desenvolvimento, usando uma loja com volume reduzido. Em uma loja real com centenas de pedidos, as tabelas ficam mais ricas — mas o formato e os campos são exatamente estes.

Como instalar e usar

Configure uma vez, use para sempre. Em ~5 minutos sua loja está conectada ao Claude.

1
Instale o Claude Code
O Claude Code é o CLI da Anthropic que roda no seu terminal e interpreta as skills. Requer Node.js 18+.
npm install -g @anthropic-ai/claude-code
Após instalar, rode claude no terminal para confirmar que está funcionando.
2
Copie as skills para a pasta do Claude
Cada skill é um arquivo SKILL.md com instruções que o Claude lê automaticamente ao ser invocado. A estrutura é ~/.claude/skills/<nome-da-skill>/SKILL.md.
# Clone o repositório git clone https://github.com/thiagorocha-2/nuvemshop-claude-skills # Copie todas as 13 skills de uma vez mkdir -p ~/.claude/skills cp -r nuvemshop-claude-skills/*/ ~/.claude/skills/
Cada pasta é independente — copie só as skills que quiser usar.
3
Obtenha as credenciais da Nuvemshop
No painel admin da sua loja: Potencializar → Aplicativos sob medida. Copie o Store ID (número na URL) e o Access Token gerado.
O token não expira — você só precisa fazer isso uma vez por loja.
4
Configure as variáveis de ambiente
As skills leem as credenciais via variáveis de ambiente. Adicione ao seu ~/.zshrc ou ~/.bashrc para não precisar redigitar.
# Adicione ao ~/.zshrc (permanente) export NUVEMSHOP_STORE_ID=seu_store_id export NUVEMSHOP_TOKEN=seu_access_token # Recarregue o terminal source ~/.zshrc
5
Abra o Claude Code e invoque qualquer skill
Na pasta do seu projeto (ou em qualquer lugar), abra o Claude Code e digite o nome da skill com /. O Claude detecta automaticamente os arquivos SKILL.md e executa o fluxo.
# No terminal claude # Dentro do Claude Code, invoque qualquer skill: /daily-digest /catalog-health /rfm-segmentation
Cada skill guia você pelo fluxo — coleta dados, exibe preview e pede confirmação antes de qualquer escrita na API.
